## 📌 About The Project

**PetCare Hub** is a modern and responsive pet care website designed to provide useful information and services for pet owners.

The website helps users explore pet care tips, pet services, health guidance, and important resources for keeping their pets happy and healthy.

---

## ✨ Features

- 🐶 Attractive pet care landing page
- 🐱 Responsive design for all devices
